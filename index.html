LangLandia Documentation Creation

I have been learning Spanish and found a game I enjoy playing that helps with vocabulary. It is called LangLandia. While playing, I noticed many people had questions about the game's nuances, but there wasn't documentation to provide answers. So I created a guide. I gathered information by collecting data myself and talking with other players. Once I compiled everything, I shared it in the game chat. The developer saw it and liked it so much that he sent out messages with the link to everyone in the game. This guide demonstrates my attention to detail and ability to create documentation, which I've also needed as an analyst when developing tools and training material for people.
